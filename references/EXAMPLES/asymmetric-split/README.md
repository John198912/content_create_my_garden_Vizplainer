# asymmetric-split —— 非对称分屏（C2 anchor）

> ⚠️ **结构示意，不是抄袭模板**。看「形」，按你的内容 + 主题换内容选型。

## 适用场景

口播在讲一个**「观点 + 论据」**对：

- 一句论断 + 一张图证明它
- 一个结论 + 一组数据支撑
- 一个主张 + 一个 mockup / 截图佐证

凡是「左边一句话，右边一个证据」的时候用它。

## 核心手段（落地 Vizplainer 留白哲学）

| 签名 | 这里怎么用 |
|---|---|
| **33% / 67% 非对称** | 左窄列放低密度文字（一句观点 + 一个 mono 标签），右宽栏放高密度视觉。**不是 50/50** —— 对称分屏是 PPT 味，非对称才有编辑/杂志感。 |
| **#6 密度反比** | 左列越空（大留白、就两三行字），右栏越实（窗口 + 图表填满）。两边密度反差本身就是构图张力。 |
| **#3 accent 预算** | 左列 accent **只点一个词**（`.as-em`）；右栏 accent 只给「赢家那根柱子」。一屏 1-2 处满格，其余 ink。 |
| **#7 mono 语境** | 标签 / 数据值用 `.sys-label` + mono；论点用 display 字体；旁白用 `.human-note`。 |

## step 切分

```
step 0  论点    左列：标签 + 一句论断浮现（右栏空）
step 1  论据    右栏窗口 + 对照柱（满格灰）浮现
step 2  收束    赢家柱（accent）刷出 + 左列补一句数字注解
```

→ narrations.ts **必须 3 条**。

## 用到的全局原语

- `.window-frame`（右栏视觉容器，animations.css §13）
- `.sys-label` / `.human-note`（base.css）
- 章节自绘柱状图（`.as-bar-*`，每根 `scaleX` 错峰长出）

## 切到其它主题怎么换

- **栅格不动**（33/67 是构图，不是主题）。
- 暗色主题：柱子赢家加 glow（`box-shadow: 0 0 24px var(--accent-glow)`）。
- 浅色编辑主题（newsroom / paper-press）：去掉窗口阴影变柔，对照柱用 `--text-faint` 浅灰即可。
- 想要更狂：右栏换成 SVG path 自绘折线 + 数字滚动（见 CHAPTER-CRAFT.md Part 3）。

## 反模式提醒

- ✗ 左右 50/50 等宽 → 失去非对称张力。
- ✗ 左列塞满五行字 → 密度反比塌掉，变成两块一样满的板。左列要敢留白。
- ✗ 两根柱子都上 accent → 对照失效。**只有赢家满格 accent**，对照项必须灰。
