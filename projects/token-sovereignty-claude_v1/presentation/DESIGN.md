# Visual Identity — Token Sovereignty

> Theme: bold-signal-v2. Hot orange pitch-deck DNA with magenta sweep accent. Full neon-glow card, ambient orbs, and cascade reveals.

## Style Prompt

焦灼、紧迫、直击痛点。像一个在危机中被放大的信号灯——橙色的热忱、玫红的警示、深黑底上文字如电报纸条般逐条弹入。信息密度高但不臃肿，每一条主张都像一个"论点卡片"被扔到观众面前。视觉节奏是"先砸结论，再给证据"。不做温和铺垫，不做模糊过渡。

## Colors

- Shell: `#0c0c0c`
- Surface: `#16161a`
- Surface-2: `#232329`
- Surface-3: `#34343c`
- Text: `#f5f5f5`
- Text-2: `#d8d8d8`
- Text-mute: `#8a8a8a`
- Accent: `#ff5722` (hot orange signal)
- Accent-2: `#d81e88` (magenta sweep — gradients/glows only, never solo)
- Rule: `#3a3a44`

## Typography

- Display CN: Noto Sans SC, weight 700 for L1 claims
- Display EN: Archivo Black, weight 900 for hero numbers and English quotes
- Body: Space Grotesk + Noto Sans SC
- Mono: JetBrains Mono — for timestamps, source labels, mono caps

## Motion

- Primary transition: directional push, 440ms smoothed ease-out
- Accent transition: focus pull for chapter reveals
- Entrance: cascade fade-in (85ms stagger), rise-step (22px lift), glow-card reveal
- Ambient: low-opacity orbs breathing orange→magenta
- No spring. Everything is smoothed cinematic ease-out.

## What NOT to Do

- No purple-pink gradients, neon particles, or meaningless equalizers
- No averaged card walls or 6-panel dashboards
- No remote fonts or undeclared assets
- Never full-screen transition between beats within same scene
- Never shrink font to fix layout — restructure instead
- Do not mix CSS and shader capture paths
- Accent-2 (magenta) is gradient/glow partner only — never standalone second brand color

## Scene-Specific Notes

- Opening: Karp CNBC capture + dramatic quote reveals — cascade cards
- Arguments 1-4: Numbered argument cards, each with numbered circle + claim + evidence
- Enterprise-to-Individual: Side-by-side mirrored comparison using shared coordinate space
- Root Cause: Data lake diagram → sharingan metaphor → signal-to-noise wave
- Global: World map simplified to three glowing signal points
- Three Principles: Clean principle cards with numbered circles, Karp correspondence badges
- Ending: Return to center — single focused message, reduced motion energy
